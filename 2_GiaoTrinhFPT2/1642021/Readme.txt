1.Hướng dẫn chạy chương trình:

- Tại thư mục chứa tập tin thực thi quanlyhocsinh.jar (../Release/):
	_B1: Mở file connectDB.xml, thay đổi hostname,databaseName,user,password(nếu không có để trống) ứng với máy đang chạy
	_B2: Thực hiện 1 trong 2 cách để chạy chương trình:
		C1: Mở cửa sổ command line(cmd),đ ứng tại thư mục chứa tập tin thực thi chạy câu lệnh: java -jar quanlyhocsinh.jar
		C2: Double click vào file quanlyhocsinh.jar
- Nếu máy chưa có database thì thực hiện chạy file scriptSQL.sql

2. Source code của bài tập được đặt trong thư mục Source
Lưu ý: Database sinh viên tạo column MaHS tự tăng nên không cần nhập mã học sinh lúc thêm mới (Có thể phát sinh lỗi nếu máy chạy mã sinh viên không tự tăng)
- Tiêu chí tìm kiếm là theo Mã hoặc Tên hoặc Ngày Sinh của học sinh
